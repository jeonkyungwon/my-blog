# Next.js 블로그 시작하기

이 글은 **Next.js + MDX** 조합으로 블로그를 만드는 프로젝트의 첫 번째 글입니다.  
TailwindCSS와 shadcn/ui도 함께 사용하며 다크모드와 카드형 글 리스트를 구성합니다.

## 소개

- 프레임워크: Next.js 14 (App Router)
- 스타일: Tailwind CSS + shadcn/ui
- 렌더링: SSG + MDX
