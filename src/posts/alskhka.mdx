## 목차 기능 소개

이 글은 ScrollSpy 기능을 실험하기 위한 테스트용 글입니다.  
스크롤에 따라 오른쪽 목차에서 현재 위치가 강조되어야 합니다.

### 왜 ScrollSpy가 필요할까?

긴 글에서는 사용자가 어디를 보고 있는지 파악하기 어렵습니다.  
이를 해결하기 위해 사용 위치를 표시해주는 기능이 필요하죠.

---

## Intersection Observer 이해하기

ScrollSpy 기능은 보통 `IntersectionObserver` API로 구현합니다.

### 기본 개념

Intersection Observer는 요소가 뷰포트에 들어왔는지 감지합니다.

### 예시 코드

```ts
const observer = new IntersectionObserver((entries) => {
  console.log(entries);
});
```
